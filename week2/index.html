JavaScript statements and operators

Syntax basics

A simple statment.

var s = "hello world";

We can ignore whitespace (except for inside the quotation marks).

var x =         " hello world ! ";

The = sign is for assigning values.

x "gets" 10

Operators

Add and subtract

var x = 10;
var y = 20;
x + y;
x - 5;
y + 5 - x;
x - (y + 5);

Multiply and divide

2 * 2;
2 / 3;

Concatenation

var h = "hello";
var w = "world";
x + w;
"hello" + "world";
"hello " + "world";

Increment, decrement

var x = 0;
x++;
x += 10;
var y = ++x;
var z = x++;
x--;
x -= 5;

Addition vs. concatenation

var x = 1;
var y = "7";
x + y;
typeof x + y;
y + x;
x + +y;
x + Number(y);
