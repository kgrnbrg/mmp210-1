p5: text

MMP 210 > Week 7

We can add text to our sketches with the text() function of p5.

text(string, x, y) requires at least 3 arguments:

  • string: the text to display
  • x: x-coordinate of text
  • y: y-coordinate of text

A simple example:

function setup() {
	createCanvas(640, 480);
	background("white");
	text("MMP 210", 20, 20);	
}

Let's review string.

A string is a datatype comprised of a "string" of characters. Characters can be alphabetic, numeric and special characters. Strings can be made of English characters as well as characters from other supported languages. Some examples:

var str1 = "Hello world.";
var str2 = "你好，世界";
var str3 = "Strings can also include numbers like 100, punctuation like !~+:) and special characters like π or © or ∆";
text(str1, 20, 20);
text(str2, 20, 40);
text(str3, 20, 60);

Strings have a bunch of methods and properties that can be used to manipulate them.

var str = "Hello world";
console.log(str.length);
console.log(str.charAt(0));
console.log(str.charAt(str.length - 1));
console.log(str[0]);
console.log(str[str.length - 1]);
console.log(str.replace("Hello", "goodbye"));

Strings can also be added together (remember our greet() function from week 5?) and compared.

var a = "a";
var b = "b";
console.log(a > b);
console.log(b > a);

var h = "hello";
var w = "world";
console.log(h + " " + w);

Styling strings with p5

p5 uses the the stroke and fill colors like other shapes, but also uses textSize(), textStyle() and textFont() to change the style of our text.

function setup() {
	createCanvas(640, 480);
	background("white");
	
	text("MMP 210", 20, 20);

	textSize(100);
	text("Week 7", 20, 150);

	textStyle(ITALIC);
	text("p5 Text", 20, 300);

	textFont("Georgia");
	text("MMP 210", 220, 20);
}

Paragraph styles are set with textWidth() and textAlign().

function setup() {
	createCanvas(640, 480);
	background("white");
	
	textWidth(width);
	textAlign(CENTER);

	var paragraph = "Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod" + "\n" +
	"tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam," + "\n" +
	"quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo" + "\n" +
	"consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse" + "\n" +
	"cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non" + "\n" +
	"proident, sunt in culpa qui officia deserunt mollit anim id est laborum.";

	text(paragraph, width/2, 20);
}
